# MOSS_DevOps_Demo

## 仓库简介

本仓库是2023年北航敏捷软件工程课程MOSS队关于DevOps的演示仓库

## 目录简介

### `Vue_CICD_Demo`

该目录下的`index.html`是本仓库用于演示的小型Vue项目的前端页面，通过每位成员在其中完成个人内容来展示CI/CD。

Vue项目的其余配置文件等均部署在服务器，没有上传到本仓库以保证仓库的简洁清晰。

### `.github/workflows`

该目录下的`main.yml`是GitHub Actions控制文件，用于设置CI/CD工作流。
